## Getting started
1. Download and Install Nodejs
2. Clone this repository:
```
---
git clone https://github.com/chungpj/chat-app.git
cd chat-app
---
```
3. Install dependencies:
```
---
npm install
or 
npm install express --save
npm i socket.io
npm install delay
---
```
4. Run
```
---
node index.js
---
```
5. Open browser: http://localhost:3000/
